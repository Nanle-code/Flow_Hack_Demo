# FlowKit Demo

Interactive demo showcasing FlowKit's cross-VM capabilities.

## Run Demo

```bash
npm run demo
```

Then open http://localhost:3000 in your browser.

## Features

- Wallet connection with COA creation
- Token transfers (FLOW, USDC, USDT)
- NFT transfers
- Cross-VM routing visualization
- Transaction log
- Live VM detection

## Demo Flow

1. Click "Connect Wallet" (mock authentication)
2. Enter recipient address
3. Select token/NFT
4. Click "Transfer"
5. See automatic VM detection and routing

The demo uses mock data but demonstrates the real FlowKit API.
