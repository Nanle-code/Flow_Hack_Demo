03 · flowkit.batch() & sponsor() — coming alive
Batch Transactions

Execute N EVM calls atomically with one wallet signature. All succeed or all revert.

await batch([ // 1. Approve USDC to vault { contractAddress: USDC_EVM, calldata: encodeApprove(VAULT, amount) }, // 2. Deposit into vault { contractAddress: VAULT_EVM, calldata: encodeDeposit(amount) } ]) // → ONE signature, TWO EVM calls // → atomic: if deposit fails, approve reverts too
⚡ Up to 10 calls
🔒 Atomic revert
✍️ 1 signature
Gas Sponsorship

Users pay zero gas fees. Your backend sponsor covers costs. New users onboard with social login and transact immediately.

// User has 0 FLOW — doesn't matter const result = await sponsor({ cadence: MY_TRANSACTION, args: (arg, t) => [arg('hello', t.String)], sponsor: '0xYourBackendSponsor' }) result.feeSponsored // true ✓ // proposer = user (their sequence #) // payer = sponsor (pays fee)
